概要：

消费一条消息往往比产生一条消息慢很多，为了防止消息积压，一般需要开启消费者多线程同时消费消息。

 

application.properties配置：

  1. spring.rabbitmq.host=localhost
  2.  
  3. # TCP/IP端口为5672，http端口为15672
  4. spring.rabbitmq.port=5672
  5.  
  6. spring.rabbitmq.username=root
  7.  
  8. spring.rabbitmq.password=root

生产者：

  1. package com.example.rabbitmq;
  2.  
  3. import org.springframework.amqp.rabbit.core.RabbitTemplate;
  4. import org.springframework.beans.factory.annotation.Autowired;
  5. import org.springframework.web.bind.annotation.GetMapping;
  6. import org.springframework.web.bind.annotation.RestController;
  7.  
  8. @RestController
  9. public class RabbitMQController {
  10.     
  11.     // 这里用的是RabbitTemplate发消息，也可以用AmqpTemplate，推荐使用RabbitTemplate。
  12.     @Autowired
  13.     private RabbitTemplate rabbitTemplate;
  14.  
  15.     @GetMapping(value = "/helloRabbit")
  16.     public String sendMQ(){
  17.         for (int i = 0; i < 100; i++) {
  18.             rabbitTemplate.convertAndSend("myExchange1", "routingKey1", "消费者多线程");
  19.         }
  20.         return "success";
  21.     }
  22. }

消费者：

  1. package com.example.rabbitmq;
  2.  
  3. import org.springframework.amqp.core.Message;
  4. import org.springframework.amqp.rabbit.annotation.*;
  5. import org.springframework.stereotype.Component;
  6.  
  7. @Component
  8. public class Receiver {
  9.  
  10.     /**
  11.      * @RabbitListener:加了该注解的方法表示该方法是一个消费者
  12.      * concurrency：并发数量。
  13.      * 其他属性和注解想了解的话，自己按Ctrl点进去看
  14.      */
  15.     @RabbitListener(
  16.             bindings = @QueueBinding(
  17.                     value = @Queue(value = "myQueue1"),
  18.                     exchange = @Exchange(value = "myExchange1"),
  19.                     key = "routingKey1"
  20.             ),
  21.             concurrency =  "10"
  22.     )
  23.     public void process1(Message message) throws Exception {
  24.         System.out.println("myQueue1:" + new String(message.getBody()));
  25.         Thread.sleep(1000);
  26.     }
  27. }

启动项目，访问http://localhost:8080/helloRabbit，控制台每隔1秒打印10条“myQueue1:消费者多线程”，打印10次。
